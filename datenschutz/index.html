Diese Website wird über GitHub Pages gehostet. Diese Daten werden ausschließlich von GitHub zur Sicherstellung des technischen Betriebs verarbeitet.

Wichtig: Ich habe keinen Zugriff auf diese Server-Logs und speichere selbst keine Besucherdaten.
