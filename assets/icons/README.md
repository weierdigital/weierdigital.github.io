# weier.digital logo package

Version: forward slash concept with parallel horizontal caps

Included:
- SVG vector files (transparent, scalable)
- PDF vector files for print/placement
- PNG files for web previews and quick use
- White background PNG variants

Core concept:
- Signet: two normal forward slashes `//`
- Slash shape: custom vector bars with parallel horizontal top and bottom caps
- Accent: small square dot in purple `#7E45C5`
- Wordmark: lowercase `weier.digital`, with `weier` slightly heavier than `.digital`
- Primary black: `#111317`

Recommended files:
- `weier-digital-logo-stacked.svg` for website/header or square-ish layouts
- `weier-digital-logo-horizontal.svg` for wide headers/footers
- `weier-digital-icon.svg` for favicon/app icon/social profile
- `weier-digital-logo-stacked-mono.svg` when one-color output is needed
